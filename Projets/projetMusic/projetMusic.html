Projets d'IA musicale :

Introduction

L'IA musicale a débuté en novembre 2024 et j'ai continué à itérer sur les versions depuis. Cette IA utilise une structure de classe comme vue à l'IUT, et j'ai respecté les conventions d'écriture de l'IUT, même si le produit est développé en Python.

Apprentissage

Ce que ce projet ma appris :

Utilisation de classe externe.

Ce projet m'a permis d'apprendre à bien utiliser et à lire la documentation de classes inconnues, notamment TensorFlow (version 1), PyTorch et Music21.

Meilleur compréhension des matrices

J'ai appris à mieux utiliser les matrices, notamment multidimensionnelles. J'ai utilisé les tenseurs (matrices à plusieurs dimensions) de PyTorch.

Qualité du code

J'ai utilisé les conventions d'écriture de l'IUT afin de mieux m'y retrouver dans le projet.
